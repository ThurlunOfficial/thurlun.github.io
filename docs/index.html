Bem-vindo ao site oficial do Thurlun (TLUN)

Uma memecoin baseada na Binance Smart Chain (BEP-20).

Sobre o Projeto

Thurlun é uma criptomoeda criada para fomentar um ecossistema inovador. Nosso objetivo é arrecadar fundos para o desenvolvimento de uma blockchain personalizada, e ao mesmo tempo proporcionar uma experiência única para nossos investidores.
