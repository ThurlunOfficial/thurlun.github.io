Bem-vindo ao Thurlun (TLUN)

Thurlun (TLUN) é uma memecoin inovadora baseada na Binance Smart Chain (BEP-20).

Sobre o Projeto

Thurlun (TLUN) foi criado para arrecadar fundos para o desenvolvimento de uma blockchain personalizada e gerar um ecossistema acessível e inovador.

O nosso objetivo é oferecer uma experiência divertida, estratégica e acessível para investidores e usuários do ecossistema cripto.
